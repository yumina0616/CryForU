# I'll Cry For You

🔗 **[Live Demo](https://yumina0616.github.io/CryForU)**

A web experience that cries for you — so you don't have to.  
My first interactive web project, built out of a love for interactive web design.

Open this whenever you need a good cry😭

---

## About

A pixelated figure weeps on your screen. Tears fall from fixed eye positions as independent particles. You can pull out a tissue from the tissue box and hold it up to catch the tears — or just let them fall.

---

## Interactions

- **⏯️ Play / Pause** — start and stop the crying
- **👁️ Blink animation** — the figure blinks at random intervals
- **💧 Tears** — fall from the eyes in varying sizes and speeds
- **📦 Tissue box** — double-click to pull out a tissue
- **🧻 Tissue** — follows your cursor; release to drop with gravity
- **🗑️ Trash can** — click or let the tissue fall into it to discard
- **👆 Icons** — drag the trash can and tissue box freely on a grid

---

## Built With

- Vanilla HTML / CSS / JavaScript
- Python + Pillow (pointillism image processing)
